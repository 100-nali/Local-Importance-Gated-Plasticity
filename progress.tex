       \large A Progress Report}

\begin{abstract}
We study catastrophic forgetting in sequential learning on coupled physical
networks. Software-side mitigations such as Elastic Weight Consolidation
(EWC) and Synaptic Intelligence (SI) reduce forgetting by computing a
per-parameter importance signal --- but they do so using global quantities
that are not available to an edge in a physical learning substrate, where
each edge can only access the potentials at its own two endpoints. The
question we ask is whether a \emph{strictly local} importance signal, with
the same one-scalar-per-edge form, can recover the retention benefits of
EWC/SI on a coupled physical network. We build a 2D resistor-network mesh
substrate with contrastive (coupled) learning and test three strictly-local
rules: vanilla contrastive learning, per-edge gradient thresholding, and a
proximal importance-gated rule. On a sequential linear regression benchmark
with controllable task overlap, the importance-gated rule Pareto-dominates
vanilla on the (past-task error, new-task error) plane: it reduces old-task
error by approximately $10\%$ while matching vanilla's new-task fit to
within noise, at the most-interference operating point. It additionally
dissipates roughly $35\%$ less heat per task. Extending to a
five-orthogonal-task sequence reveals a Pareto bound: only an
online-EWC variant (cumulative per-task importance) crosses the
no-information past-task MSE floor among the four local-scalar
importance variants we test, and three further variants (centroid
anchor, Synaptic Intelligence importance, per-task anchors via
closed-form running sums) all trace the same frontier --- suggesting
the achievable retention on this substrate is bounded by the
single-scalar-per-edge structure of the protection signal itself.
We report the simulator, the rules, the headline results from both
regimes, and a focused plan for the remaining experiments.
\end{abstract}

\section{Introduction}

A learning system that trains on a sequence of tasks tends to overwrite the
parameters that mattered for earlier tasks when it adapts to later ones.
This is \emph{catastrophic forgetting} \cite{mccloskey1989}, and it is the
central obstacle to continual learning in any system that re-uses its
parameters across tasks. The dominant deep-learning mitigations are
\emph{replay} (revisit data from past tasks) and \emph{importance-weighted
regularization} (slow down updates on parameters that mattered for past
tasks). EWC \cite{kirkpatrick2017} and SI \cite{zenke2017} are the canonical
examples of the latter family.

These methods compute an importance signal \emph{per parameter}, but the
quantities they use --- the diagonal of the Fisher information matrix, or
the path integral of the gradient along the training trajectory --- are
derived from access to the global loss landscape. In a software
implementation this distinction is invisible; in a physical learning
substrate it is the central concern. A resistor network or a mechanical
coupled lattice does not have global access to the loss: each edge can
only measure quantities at its own two endpoints. Any importance signal
that hopes to be physically implementable must respect this constraint.

Recent work on physical learning \cite{stern2022, stern2024,
thresholdedphysical2025} has shown that simple local rules --- coupled
learning, equilibrium propagation, thresholded force updates --- can
already produce striking learning behaviour. But sequential learning on
these substrates still exhibits forgetting, and the literature has not yet
demonstrated a strictly-local rule that recovers EWC/SI-style retention
benefits. \textbf{This is the gap this project aims to close}: build a
strictly-local importance-gated rule, on a physically realisable
substrate, and test whether it can match the canonical
importance-regularised retention benefits without using any global
information.

The progress reported here covers the simulator, the rule design, and the
first round of empirical results. The headline finding is that on the
substrate we use, the proposed rule Pareto-dominates the natural baseline
(vanilla coupled learning) on the (past-task error, new-task error) plane:
it improves retention without paying for it in plasticity.

\section{Background}

\subsection{Coupled physical networks}

A linear coupled network is an undirected weighted graph in which each
node has a real-valued potential $v_i$ and each edge $e = (i, j)$
contributes $\tfrac{1}{2} w_e (v_i - v_j)^2$ to a quadratic energy $E$.
Some nodes are \emph{clamped} to fixed values (the network's inputs);
the others equilibrate to the unique minimum of $E$ given those clamps
(for positive $w_e$). Writing $L$ for the weighted graph Laplacian and
partitioning into free $f$ and clamped $c$ blocks, the equilibrium
satisfies
\[
L_{ff}\, \mathbf{v}_f = -L_{fc}\, \mathbf{v}_c.
\]
The substrate's forward computation is this linear system solve. With
positive conductances this corresponds physically to a passive resistor
network; signed couplings would correspond to a mechanical-spring system
with both attractive and repulsive springs.

\subsection{Contrastive (coupled) learning}

Coupled learning \cite{stern2022, stern2024} and equilibrium propagation
\cite{scellier2017} both express the gradient of an output-based loss
with respect to an edge weight as a local energy difference between two
equilibria. The \emph{free} phase $\mathbf{v}^F$ uses only the input
clamps; the \emph{clamped} phase $\mathbf{v}^C$ additionally nudges the
output toward the target with strength $\eta$. The per-edge gradient is
\begin{equation}
\label{eq:cl}
g_e = \frac{1}{2\eta}
\left[
(v^C_i - v^C_j)^2 - (v^F_i - v^F_j)^2
\right],
\end{equation}
which approaches $\partial L / \partial w_e$ in the limit $\eta \to 0$.
Crucially, each edge needs only the potentials at its own two endpoints in
both phases. There is no chain rule, no backpropagation, no global
gradient computation.

\subsection{Diagonal Fisher information as importance}

EWC's importance proxy is the diagonal of the empirical Fisher information,
$F_{ii} = \mathbb{E}[(\partial L / \partial w_i)^2]$. The penalty added to
the loss during a new task is
$\tfrac{c}{2}\sum_i F_i (w_i - w_i^*)^2$, anchoring each parameter to its
end-of-previous-task value $w_i^*$ with stiffness proportional to its
own Fisher entry. The strictly-local analogue we use replaces the Fisher
diagonal by a per-edge exponential moving average of $g_e^2$ --- a
quantity each edge can compute from its own gradient history --- and the
penalty by a proximal (implicit) update toward $w_e^*$ with
per-edge stiffness $\lambda I_e$. We give the explicit update in
Section~\ref{sec:methods}.

\section{Methods}
\label{sec:methods}

\subsection{Substrate: 2D resistor-network mesh}

The substrate is an $8 \times 10$ rectangular grid of nodes with
4-neighbour (N, S, E, W) connectivity. There are 142 trainable edges
($\text{rows} \times (\text{cols} - 1) + (\text{rows} - 1) \times \text{cols}$).

\begin{itemize}
\item Inputs are clamped at the leftmost column, rows $0..7$, to the
      eight input features $x_0..x_7$.
\item The output is the differential pair on the rightmost column,
      $\hat y = v[\text{out}_+] - v[\text{out}_-]$, with rows $3$ and $4$
      taken as the $\pm$ output nodes.
\item All other nodes are free.
\end{itemize}

Conductances are strictly positive ($w_e \geq w_{\min} = 0.05$), making
this a passive resistor network. The differential output is what allows
a positive-conductance system to fit signed regression targets without
resorting to signed couplings.

\subsection{Three local learning rules}

All three rules use only quantities visible at the edge --- the per-edge
contrastive gradient $g_e$ from Eq.~\ref{eq:cl} plus whatever scalar
state the rule maintains at that edge.

\paragraph{Vanilla coupled learning.}
The baseline rule, pure unconstrained contrastive learning:
\[
w_e \leftarrow w_e - \mathrm{lr} \cdot g_e.
\]
No per-edge state, no protection.

\paragraph{Threshold.}
The local-force-thresholding rule of \cite{thresholdedphysical2025},
applied to the per-batch contrastive force:
\[
w_e \leftarrow
\begin{cases}
w_e - \mathrm{lr} \cdot g_e & \text{if } |g_e| > \tau,\\
w_e & \text{otherwise.}
\end{cases}
\]
The threshold gates plasticity: only edges with above-threshold force
update. No per-edge state beyond the threshold value $\tau$ itself.

\paragraph{Importance-gated.}
Each edge maintains an anchor $w^*_e$ (snapshot of $w_e$ at the previous
task boundary) and an importance $I^*_e$ (the per-edge $\beta$-EMA of
$g_e^2$, also snapshotted at the previous task boundary). The update
during a new task is the proximal (implicit) step
\begin{equation}
\label{eq:proximal}
w_e \leftarrow
\frac{w_e - \mathrm{lr} \cdot g_e + \mathrm{lr} \cdot \lambda \cdot I^*_e \cdot w^*_e}
     {1 + \mathrm{lr} \cdot \lambda \cdot I^*_e}.
\end{equation}
This is algebraically the implicit Euler step on the augmented loss
$L + \tfrac{1}{2} \lambda \sum_e I^*_e (w_e - w^*_e)^2$. The denominator
is always $\geq 1$, so the rule is unconditionally stable regardless of
how large $\lambda I^*_e$ becomes; aggressive $\lambda$ values do not
require gradient clipping. The first task uses plain contrastive
learning (no anchor yet). At each task boundary, $w^*_e \leftarrow w_e$
and $I^*_e \leftarrow I_e$ (where $I_e$ is the live EMA tracked over the
just-completed task).

The rule depends only on $w_e, w^*_e, I_e, I^*_e, g_e$ --- all per-edge.
No global Fisher computation, no cross-edge coupling, no replay buffer.

\subsection{Sequential linear regression with controllable task overlap}

A task is a linear regression with target direction
$\mathbf{d} \in \mathbb{R}^8$, $|\mathbf{d}| = 1$. Inputs are
$\mathbf{x} \sim \mathcal{N}(0, I_8)$, targets are
$y = \mathbf{d} \cdot \mathbf{x} + \varepsilon$ with
$\varepsilon \sim \mathcal{N}(0, 0.05^2)$. Two consecutive tasks $A$ and
$B$ are constructed so that $\mathbf{d}_A \cdot \mathbf{d}_B = \rho$ where
$\rho \in [0, 1]$ controls task interference. $\rho = 1$ produces identical
tasks (no interference); $\rho = 0$ produces orthogonal target directions
(maximum interference). Each task has 500 training and 200 test samples.
We sweep $\rho$ at 21 evenly spaced points in $[0, 1]$ with 12 random
seeds per cell.

\subsection{Metrics}

For each rule and each overlap, after training $A$ then $B$:

\begin{itemize}
\item \textbf{Past-task error after $B$} (test MSE on task $A$'s data
      after training on $B$) --- the primary forgetting metric.
\item \textbf{New-task error after $B$} (test MSE on task $B$'s data) ---
      the primary plasticity metric.
\item \textbf{Heat dissipated}: per-edge work $Q_e = -g_e \Delta w_e$
      summed over training steps and edges. For vanilla this reduces
      to the canonical viscous dissipation $\mathrm{lr} \sum_t g_t^2$;
      for the importance-gated rule it includes both the damped gradient
      term and the anchor-pull cross term. Reported as median over seeds
      because the seed distribution is broad.
\item \textbf{Spatial overlap of edge activity}: cosine similarity,
      Spearman rank correlation, Jaccard of top-20\% edges, and mass
      overlap of the per-task $|\Delta w|$ distributions.
\end{itemize}

\subsection{Pareto improvement over vanilla coupled learning}

The key empirical result is the Pareto curve in Figure~\ref{fig:pareto}.
At the most-interference operating point ($\rho = 0$), a sweep of the
importance-gated rule over $\lambda$ traces a curve on the
$(A_{\text{after }B}, B_{\text{after }B})$ plane that passes below
vanilla coupled learning's operating point. At $\lambda = 100$

$A = 1.332$ --- a $10.5\%$ retention gain --- while matching vanilla's
$B = 0.628$ within the seed noise ($B = 0.626$). At smaller $\lambda$
($\in \{20, 30, 50\}$) the rule pushes \emph{below} vanilla's $B$ as
well, strictly dominating vanilla on both axes simultaneously.

\subsection{Thermodynamic dissipation}

The heat metric $Q_e = -g_e \Delta w_e$, summed over edges and steps,
measures the work the loss-landscape force performs on the network
during a task. At $\rho = 0$ the median heat per task is approximately
$30.9$ for \textsf{vanilla}, $3.6$ for \textsf{thresh}, and $20.2$ for
\textsf{imp\_gated}: the importance-gated rule dissipates roughly $35\%$
less than vanilla while retaining more old-task memory.
Figure~\ref{fig:heatpareto} plots retention against median heat
dissipated at each overlap. The \textsf{imp\_gated} markers sit
above-and-left of the \textsf{vanilla} markers at every overlap ---
strict Pareto dominance on the (heat, retention) plane.
\textsf{thresh} sits at low heat but at low retention; it trades
dissipation for plasticity, not for retention.

\subsection{Five-task orthogonal sequence: a Pareto bound for local-scalar importance}
\label{sec:multitask}

The two-task results above establish that the proximal importance-gated
rule retains past-task knowledge while matching vanilla's new-task fit.
The next question is whether the benefit scales. We construct five
mutually orthogonal tasks --- the directions $\mathbf{d}_1, \dots,
\mathbf{d}_5$ are the first five columns of a Haar-random orthonormal
matrix in $\mathbb{R}^8$ --- and train them sequentially. Orthogonality
is the maximum-interference regime: each new task's gradient is
uncorrelated with every past task's, so no inter-task sharing can mask
poor retention.

\paragraph{The no-information floor.}
For unit-norm regression targets $y = \mathbf{d}_k \cdot \mathbf{x}$
with $\mathbf{x} \sim \mathcal{N}(0, I)$, an output of zero yields
$\mathbb{E}[(0 - y)^2] = \mathbb{E}[y^2] = 1$. A network with
$\text{MSE} > 1$ on task $k$ is therefore worse than uninformative on
that task: it has actively learned a different output. We refer to
$\text{MSE} = 1$ as the \emph{no-information floor} and use it as the
reference for genuine retention --- a useful sharpening of ``did not
catastrophically forget,'' which can be satisfied trivially by not
learning anything.

\paragraph{Snapshot importance fails to scale.}
At five tasks, all three rules of Section~\ref{sec:methods} stay above
the no-information floor on past-task MSE
(Table~\ref{tab:multitask-headline}, top rows). Vanilla coupled
learning's past-mean MSE is $1.43$ --- it has overwritten past-task
directions with the most recent. \textsf{thresh} sits near the floor at
$1.04$, but its current-task MSE is also $0.96$: it is not retaining,
it is under-learning. \textsf{imp\_gated} at $\lambda = 100$ (the
operating point from the two-task Pareto) is intermediate: past-mean
$1.28$, current $0.63$. The structural cause is that the rule's
snapshot $I^*_e \leftarrow I_e$ overwrites previous-task importance at
every boundary, so at task 5 each edge only remembers importance
accumulated during task 4 --- protection against the
immediately-previous task, not against any earlier ones.

\paragraph{Cumulative (online-EWC) importance crosses the floor.}
The simplest fix is to accumulate $I^*$ across boundaries rather than
replace it: $I^*_e \leftarrow I^*_e + I_e$. Per-edge memory stays at a
single scalar; the proximal step is unchanged. We call the resulting
rule \textsf{cum\_imp\_gated}. With this one-line change, at
$\lambda = 500$ the rule achieves past-mean $0.992 \pm 0.013$ and
current $0.944 \pm 0.043$ over $12$ seeds --- the first point on the
(past, current) plane that falls below the no-information floor on both
axes simultaneously (Table~\ref{tab:multitask-headline},
Figure~\ref{fig:multitaskpareto}). \textbf{\textsf{cum\_imp\_gated} is
the only rule of the four we tested in this regime that genuinely
retains past tasks rather than merely interfering with them less.}

\paragraph{Anchor location and importance estimator do not change the frontier.}
Two natural strengthenings of \textsf{cum\_imp\_gated} do not move the
Pareto curve. First, the single anchor at the most-recent
end-of-task weights is replaced by the importance-weighted centroid of
all past anchors. The proximal step's closed form requires only two
running sums per edge,
\begin{equation}
\label{eq:SR}
S_e = \sum_k I^*_{e,k}, \qquad R_e = \sum_k I^*_{e,k} w^*_{e,k},
\end{equation}
so per-edge memory stays at two scalars regardless of task count, and
the update becomes
\[
w_e \leftarrow \frac{w_e - \mathrm{lr}\,g_e + \mathrm{lr}\,\lambda\,R_e}
                    {1 + \mathrm{lr}\,\lambda\,S_e}.
\]
We call this rule \textsf{multi\_anchor}: equivalent to maintaining a
separate quadratic penalty per past task, but with closed-form
constant-memory bookkeeping. Second, the EMA-of-$g_e^2$ importance
estimator is replaced by the substrate-native Synaptic Intelligence
estimator \cite{zenke2017},
$I_e \leftarrow \sum_t \max\!\big(0,\, -g_{e,t}\,\Delta w_{e,t}\big)$
--- the cumulative work the contrastive force did on this edge during
the task, equivalently the heat $Q_e$ we already track end-to-end. SI
typically outperforms Fisher-diagonal on orthogonal continual
sequences. Neither change moves the Pareto curve: across $\lambda$
sweeps with $12$ seeds each, the three cumulative variants produce
statistically indistinguishable frontiers that saturate at
$(\text{past}_{\text{best}}, \text{current}_{\text{best}}) \approx
(0.98,\, 0.95)$. On Figure~\ref{fig:multitaskpareto} the
\textsf{multi\_anchor} and \textsf{heat\_cum\_imp\_gated} traces sit
inside the seed-noise envelope of the basic cumulative rule across
every $\lambda$ value we tested.

\paragraph{Implication: the bound is structural.}
Three orthogonal modifications --- anchor location, importance
accumulation policy, and importance estimator --- do not improve the
achievable retention. The feature they share is the constraint we have
imposed throughout: each edge holds a single scalar importance, and the
anchor pull is isotropic in parameter space. A scalar can record that
an edge \emph{mattered} for past tasks, but it cannot record
\emph{which} input-space direction the edge was serving. On
orthogonal tasks the same edges have to serve different input-space
directions across tasks; scalar importance cannot disambiguate them,
so a single operating-point weight is right for at most one task at a
time. The Pareto bound near $\text{MSE} = 1$ is the empirical signature
of this constraint. Breaking it appears to require non-scalar per-edge
state --- for example, a direction-of-importance derived from
clamped/free voltage differences, which the substrate already exposes
edge-locally --- at the cost of stepping outside the
single-scalar-per-edge family that has been our locality criterion so
far.

\section{Discussion}

The empirical picture so far is consistent across the three result
panels. On forgetting alone, the importance-gated rule sits below the
two baselines at every overlap (Figure~\ref{fig:overlap}). On the
plasticity--stability trade-off, it Pareto-dominates the vanilla
baseline --- improving past-task retention without paying a
new-task-fit cost at the operating point (Figure~\ref{fig:pareto}). On
heat dissipation, it sits on a strictly better (heat, retention) frontier
than vanilla (Figure~\ref{fig:heatpareto}). And the spatial visualisation
suggests that the underlying mechanism is allocation: the snapshot
importance localises the protection to edges that mattered for task $A$,
and the new task is routed through the rest of the mesh
(Figure~\ref{fig:zones}, Table~\ref{tab:spatial}).

Two interpretive caveats are worth flagging. First, the per-seed
distribution on this substrate is broad: the coupled-network loss is
non-convex in the edge weights and some seeds settle into local minima.
We mitigate by reporting medians on the heat metric and by using 12
seeds per cell, but the IQRs visible in Figure~\ref{fig:overlap} are not
small. Second, the heat metric is computed from the batch-averaged
contrastive force; an online (sample-by-sample) physical implementation
would dissipate more heat by a minibatch-variance term. The relative
ordering between rules is preserved (the variance term is a property of
the data, not the rule), but the absolute heat values are a lower bound
on the physical dissipation.

The spatial separation visible in Figure~\ref{fig:zones} is real but
not dramatic. The single-output bottleneck in our substrate means all
signal flows through a narrow neighbourhood near the output diamonds,
which puts a structural ceiling on how disjoint the per-task active
edge sets can be. A substrate with multiple output pairs, or with sparse
random connectivity, would give the rule more capacity to allocate
tasks to distinct geographic regions.

\section{Further work}

\paragraph{Global EWC/SI baseline.}
The most important missing experiment. Implement EWC with a global
Fisher-diagonal estimate (computed over the full training set at task
boundaries) on the same substrate, and quantify the gap to the local
\textsf{imp\_gated}. If the gap is small, the locality constraint is
essentially free; if large, it sets a target for further rule design.

\paragraph{Beyond local scalar importance.}
Section~\ref{sec:multitask} establishes that four variants of
local-scalar-importance rules --- snapshot vs.\ cumulative,
single-anchor vs.\ centroid, Fisher-diagonal vs.\ Synaptic-Intelligence
estimator --- all converge to the same Pareto frontier on a five-task
orthogonal sequence, saturating at past-task MSE just below the
no-information floor. The natural next move is non-scalar per-edge
state. The substrate already exposes the clamped-versus-free voltage
difference at each edge --- a task-specific local activity signature
--- so a directional importance, e.g.\ a per-edge tangent vector along
which the edge ``points'' for the current task, is constructible
without breaking edge-locality. Two rules to try: (i) a per-edge
$2$-vector $(I_e^\parallel, I_e^\perp)$ with separate stiffness in the
past-task direction and orthogonal directions; (ii) replacing scalar
$I_e$ by a covariance-style estimate built from voltage differences at
the edge over the course of the task. Whether either breaks the
$(0.98, 0.95)$ bound is the next open empirical question.

\paragraph{Longer sequences and varied overlap.}
The five-task experiment used orthogonal directions; the general
sequential-overlap analogue would sweep an
$N \times N$ pairwise-overlap matrix at $N = 5, 10$ and ask whether
the Pareto bound also depends on $N$ or on the spectrum of the
overlap matrix. If the bound is genuinely set by scalar locality and
not by $N$, the curves should overlay across $N$.

\paragraph{Langevin dynamics and proper thermodynamics.}
Add Gaussian noise to the per-step update,
$\Delta w_e \leftarrow \Delta w_e + \sqrt{2 T \mathrm{lr}}\, \xi_e$,
making the dynamics stochastic. This admits proper analysis under the
fluctuation-theorem framework \cite{sekimoto2010, seifert2012}: heat
dissipated to the bath, work done by the importance potential, and
entropy production all become well-defined. Principled bounds on the
retention/dissipation trade-off should follow.

\paragraph{Online-equivalent heat.}
The substrate already exposes per-sample contrastive forces during the
\texttt{backward} step. Computing $\mathbb{E}_b[g_e^{(b)\,2}]$ alongside
the batch-averaged $\bar g_e^2$ is a two-line addition and would let us
report a sample-by-sample heat estimate that does not assume batched
dynamics.

\paragraph{Topology variations.}
Test the rule on (i) a sparse random graph of comparable node count,
(ii) a mesh with multiple differential output pairs (genuine geographic
separation possible), (iii) a 3D lattice. The topology dependence of the
rule's behaviour is itself an interesting question --- particularly
whether the importance-gated rule benefits more from substrates that
support spatial task allocation.

\paragraph{Hardware non-idealities.}
Add memristor-style noise and drift, finite-precision conductance, and
update quantisation to test whether the rule degrades gracefully under
hardware-realistic constraints. The proximal step's unconditional
stability should help here, but this needs verification.

\paragraph{Direct comparison to \cite{thresholdedphysical2025}.}
Reproduce the experimental setup of the thresholding paper as closely
as possible, and compare \textsf{imp\_gated} to their thresholding
result on identical benchmarks. This is the most direct test of whether
the rule recovers or exceeds the published thresholding result on the
substrate the paper actually used.
